ReleasePolicy

Description:

public enum ReleasePolicy

Release policy determines what happens with object freed by Policy.TRY_RELEASE and Policy.RELEASE.

Content:

Enum values:

  • HELPER_THREAD - Libgee spawns helper thread to free those elements. This is default.
  • MAIN_LOOP - Libgee uses GLib main loop. This is recommended for application using GLib main loop.
