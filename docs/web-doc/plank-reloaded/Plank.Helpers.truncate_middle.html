truncate_middle

Description:

public string truncate_middle (string str, int max_length)

Truncates a string at the middle to ensure it is no longer than max_length. If the string is truncated, an ellipsis will be inserted in the middle.

Parameters:

str

The string to truncate

max_length

Maximum length for the returned string

Returns:

A string not longer than max_length
