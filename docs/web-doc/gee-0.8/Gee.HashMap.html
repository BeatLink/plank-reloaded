HashMap

Object Hierarchy:

Description:


public class HashMap<K,V> : AbstractMap<K,V>

Hash table implementation of the Map interface.

This implementation is better fit for highly heterogeneous key values. In case of high key hashes redundancy or higher amount of data prefer using tree implementation like TreeMap.

See also:
