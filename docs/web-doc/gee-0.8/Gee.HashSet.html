HashSet

Object Hierarchy:

Description:


public class HashSet<G> : AbstractSet<G>

Hash table implementation of the Set interface.

This implementation is better fit for highly heterogeneous values. In case of high value hashes redundancy or higher amount of data prefer using tree implementation like TreeSet.

See also:
